# プレースタイルの可視化と戦術的フィット：データドリブンなスカウティングへの応用
**Data-Driven Scouting Project: Tactical Fit Analysis of Right Backs**

## 📌 プロジェクトの概要
2022 FIFAワールドカップのデータを用い、サイドバックの攻撃指標を定量化・可視化した分析レポートです。
「単にスタッツが高い選手を獲る」という属人的な評価から脱却し、チーム戦術と選手のデータ特性（プロファイル）を掛け合わせた「定量的フィット感」を重視したスカウティング手法を提案します。

## 🎯 現場への提言
1. **スカウティング・移籍戦略の最適化:** チーム戦術（ポゼッション重視か、ターゲットマンがいるか等）と選手のデータ特性を照らし合わせ、ミスマッチのない論理的な獲得判断を可能にする。
2. **論理的な育成指針:** 選手の数値特性を可視化することで、指導者の感覚に頼らない「戦術的リスク管理」や「強みを活かす配置」の指導を展開する。
