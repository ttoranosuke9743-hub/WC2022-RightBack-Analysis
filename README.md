# プレースタイルの可視化と戦術的フィット：データドリブンなスカウティングへの応用
**Data-Driven Scouting Project: Tactical Fit Analysis of Right Backs**

## 💡 検証したコア仮説
**「サイドバックの攻撃参加は、本当にチームの勝利に貢献するのか？」**

現代サッカーにおいてサイドバックの攻撃参加は必須とされています。本プロジェクトでは、私自身が右サイドバックとしてプレーする中での実体験を起点に、この「定説」を2022 FIFAワールドカップのデータを用いて定量的に検証しました。

## 📌 結論のエグゼクティブ・サマリー
分析の結果、**「攻撃参加＝勝利」という単純な仮説は棄却されました。**
現代サッカーにおいて優れたサイドバックの画一的な基準は存在せず、**最適解は完全にチーム戦術に依存する**という結論に至りました。

* **スタッツは戦術システムに規定される:** 仏代表クンデの攻撃指標が著しく低いのは能力不足ではなく、左サイド（エンバペ・テオ）の攻撃力を最大化するための「戦術的なリスク管理（疑似3バック化）」の表れであることを証明しました。
* **独自の「攻撃参加指数」による可視化:** アクションの頻度とゴールへの寄与度をもとに独自の指数を定義。優勝国アルゼンチンのモリーナは突出した攻撃数値を持たず、攻守の「バランス」が勝利の鍵であったことが確認されました。

*※データの前処理、各選手（ハキミ、トリッピアー等）の詳細なプロファイル、独自の指数定義プロセスはノートブック（`WC2022_RightBack.ipynb`）にて解説しています。*

## 🎯 現場・ビジネスへの提言（データドリブンな意思決定の社会実装）
本分析から得られた知見は、「単にスタッツが高いから獲得する」という属人的な評価から脱却し、以下の現場課題の解決に応用できます。

1. **戦略的マッチング（獲得・移籍判断の最適化）**
   ポゼッション志向か、ターゲットマンがいるかなど、自チームの戦術環境と選手のデータ特性（数値）を掛け合わせることで、ミスマッチのない論理的でROIの高い獲得判断を実現します。
2. **ユース世代からの論理的育成（戦術的フィットの意識）**
   指導者の感覚や精神論に頼るのではなく、「君の前進パスの数値はポゼッション戦術で大きな武器になる」といったデータに基づくフィードバックを実装。将来の高度な戦術システムに対応できる人材育成の基盤とします。

## 📖 ノートブック（WC2022_RightBack.ipynb）の構成
1. **はじめに:** 研究背景と目的（自身のプレー経験からの課題意識）
2. **分析対象:** W杯上位進出国5名の右SBとデータソース (StatsBomb)
3. **分析指標および集計結果:** 5選手のプレースタイルの定量化と「攻撃参加指数」の定義
4. **【深掘り分析】戦術的非対称性の証明:** フランス代表の左右SB（クンデ vs テオ）の比較
5. **結果と現場への提言:** 仮説の棄却と、パッションとデータの融合について

## 🛠 使用技術・データソース
* **Language:** Python
* **Data Processing & Calculation:** pandas, numpy
* **Visualization:** mplsoccer, japanize-matplotlib
* **Data Acquisition:** statsbombpy (StatsBomb APIを用いたイベントデータの自動抽出)
* **Data Source:** StatsBomb Open Data (2022 FIFA World Cup / Match & Event Data)
